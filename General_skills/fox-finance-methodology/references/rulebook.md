# 量化方法论规则书

> 来源：38条已验证、已视觉增强的视频笔记。用途是复现/审计该UP主的技术分析方法，不构成投资建议。每条规则都应输出为「触发条件 → 量化判定 → 行动 → 失效条件」。

## 0. 总原则

1. **先看风险，再看机会**：趋势结构破坏、高位鱼尾、双趋势线压制、抛物线末端、熊旗、M2/指数历史顶背离等风险信号，优先级高于KDJ/MACD金叉。
2. **先大周期，再小周期**：周线结构决定战略，日线决定交易，4H/小时线只用于细化买卖点。
3. **先结构，后指标**：箱体、通道、趋势线、斐波那契、隧道均线先定位置；KDJ/MACD/布林带做确认。
4. **只买两类**：买在足够低（结构支撑/箱体底/周线支撑/隧道支撑）或买在足够强（突破站稳、压力变支撑、均线多头支撑）。中间位置默认等待。
5. **分批而非梭哈**：尤其是ETF、周期股、有色/化工/钢铁，必须分批建仓，用ETF分散个股风险。

## 1. 风险否决规则

### RISK-01 双趋势线压制
- **触发条件**：价格在两条趋势线/切线之下运行。
- **量化判定**：日/周线连接至少两个高点/低点形成趋势线；若价格收盘同时位于两条线下方，风险显著升高。
- **行动**：回避多头或降仓；等待突破上方切线并回踩站稳。
- **失效条件**：放量突破上方趋势线，且回踩不破。

### RISK-02 长期趋势线多次受压
- **触发条件**：2-3年以上周线趋势线被反复测试。
- **量化判定**：同一长期趋势线被≥3次触及后仍受压回落，视为大跌/变盘预警；若第4次强力突破并站稳，则压力失效。
- **行动**：未突破前回避高位多头；突破后再看压力变支撑。
- **失效条件**：强力放量突破并回踩确认。

### RISK-03 抛物线末端不追
- **触发条件**：价格上涨角度越来越陡，呈抛物线式加速。
- **量化判定**：连续加速上涨、斜率明显大于前一阶段，且远离均线/布林中轨。
- **行动**：不追高，警惕反方向抛物线下跌；已有仓位考虑减仓。
- **失效条件**：加速后横盘消化并重新站稳关键均线/通道。

### RISK-04 鱼尾行情不碰
- **触发条件**：主升浪已过，标的处于高位尾声。
- **量化判定**：位置接近通道上轨/历史压力，高波动但上涨空间小，风险收益比差。
- **行动**：不追高，不做中长线；只允许短线且要有明确止损。

### RISK-05 趋势未反转只建底仓
- **触发条件**：价格靠近支撑但下降趋势尚未反转。
- **量化判定**：未突破下降趋势线；反弹后仍可能破新低。
- **行动**：只建底仓/轻仓试探；等反弹回踩不破新低再加大仓位。
- **失效条件**：再次跌破新低或关键支撑。

### RISK-06 M2/指数历史大顶背离
- **触发条件**：美股/纳指处于历史级高位。
- **量化判定**：纳指价格倍数与M2货币供应倍数高度对应历史顶（视频例：2000年纳指高点与M2到当前均约4.65倍）。
- **行动**：不要追多，回避风险；必须等结构重新确认。

### RISK-07 熊旗
- **触发条件**：急跌后出现小幅斜向上横盘/反弹。
- **量化判定**：大跌后形成贴近急跌角度的斜面，横盘2-3天后未能放量突破。
- **行动**：视为续跌结构，避免追反弹。
- **失效条件**：强力突破旗形上沿并回踩不破。

## 2. 结构与支撑压力规则

### STRUCT-01 箱体交易
- **触发条件**：周线/日线形成水平震荡箱体。
- **量化判定**：箱体上下沿由多次高低点确认；下沿为支撑，上沿为压力。
- **行动**：下沿不破可低吸/持有，上沿减仓；跌破箱底则看下一级支撑。
- **失效条件**：有效跌破箱底或放量突破箱顶。

### STRUCT-02 平行通道
- **触发条件**：价格在上升/下降平行通道内。
- **量化判定**：至少两个低点连下轨，平移至高点形成上轨；上轨压力，下轨支撑。
- **行动**：上升通道下轨买/加仓，上轨减仓/卖出；跌破下轨转空。
- **失效条件**：放量突破上轨进入加速，或跌破下轨趋势反转。

### STRUCT-03 趋势线突破回踩
- **触发条件**：价格突破关键趋势线或高点连线。
- **量化判定**：日线收盘站在线上；回踩趋势线不破。
- **行动**：确认后持有/加仓；未回踩确认不追。
- **失效条件**：日收线跌回趋势线下方。

### STRUCT-04 支撑压力转换
- **触发条件**：价格突破压力或跌破支撑。
- **量化判定**：突破压力并站稳→压力变支撑；跌破支撑→支撑变压力。
- **行动**：回踩不破原压力可买；反抽不过原支撑要减仓/回避。

### STRUCT-05 压力位「不过三」
- **触发条件**：同一压力位被多次测试。
- **量化判定**：第1-3次测试仍有压力；通常验证3次后压力失效，第4次突破概率提高。
- **行动**：第3次后关注突破；突破并站稳后按压力变支撑处理。
- **失效条件**：第3/4次仍被强压并放量下跌。

### STRUCT-06 关键点位分级
- **触发条件**：价格接近已标注关键点位。
- **量化判定例**：上证跌破3930→看3880；跌破3880→看3760。创业板3970守住→高位箱体，跌破→3930→2790附近。
- **行动**：跌破一级看下一级，不提前假设支撑有效。

### STRUCT-07 多工具共振支撑/压力
- **触发条件**：同一价格区同时出现趋势线、通道边界、斐波那契位、影线支撑带或隧道均线。
- **量化判定**：至少2类独立工具在同一窄区间重合；若还伴随前高/前低或周线影线带，支撑/压力等级上调。
- **行动**：共振支撑处可作为二次布局/观察区；共振压力处不追高，等待突破站稳。
- **失效条件**：日线或周线收盘有效跌破支撑带/突破压力带后，原判断作废。

### STRUCT-08 周线影线宽度支撑带
- **触发条件**：大周期快速回落后，价格接近前期周线密集影线区域。
- **量化判定**：取连续1-2根周线影线的上下边界作为区间，而不是单一点位；视频例为约4400-4300支撑带。
- **行动**：价格进入支撑带并止跌时可观察反弹；若仅触及区间上沿但未站稳，等待确认。
- **失效条件**：周线收盘跌破影线支撑带下沿。

## 3. 斐波那契规则

### FIB-01 国际资产0.618趋势定义
- **适用**：黄金、白银、外汇、原油等国际资产。
- **量化判定**：上涨趋势回调不能跌破0.618；下跌趋势反弹不能涨破0.618。跌破/涨破意味着原趋势被破坏。
- **行动**：0.618守住按原趋势；破0.618必须重估方向。

### FIB-02 国内资产0.382牛市分水岭
- **适用**：A股/港股板块、ETF等国内资产。
- **量化判定**：牛市结构中0.382更常用；不破0.382通常视为健康牛市回调。
- **行动**：0.382附近出现支撑可观察买点；跌破则降低趋势强度。

### FIB-03 画线必须含影线
- **量化判定**：最高点/最低点必须用上下影线，不只用实体。
- **行动**：重新画线时若忽略影线，所有支撑压力位作废。

### FIB-04 扩展压力位
- **量化判定**：1.236、1.382作为反弹/主升扩展压力带；100%扩展位也是关键目标。
- **行动**：进入扩展压力区后减仓或等待突破站稳。

### FIB-05 黄金ABC/B浪
- **触发条件**：黄金出现ABC三浪调整。
- **量化判定**：B浪反弹阻力常在0.618-0.786；视频例为4240-4300。0.618（约4085）跌破则上涨趋势破坏。
- **行动**：B浪阻力区不追；破0.618看C浪下探目标。

## 4. EMA与隧道规则

### EMA-01 短期EMA参数
- **参数**：EMA8、EMA13、EMA21、EMA55、EMA83。
- **判定**：EMA8/13短线支撑/压力；EMA21中期关键；EMA55中期压力/支撑；EMA83长期参考。
- **行动**：站上EMA21并多头排列看强；回踩EMA8/13不破可加仓；到EMA55附近注意压力。

### EMA-02 8/13EMA分批建仓
- **触发条件**：标的处于上升趋势中，价格回调到EMA8-EMA13区间。
- **量化判定**：视频例有色ETF回调到EMA8约1.734，EMA13约1.730之间。
- **行动**：分批买入，不追高。
- **失效条件**：跌破通道下轨或趋势由上升转下降。

### EMA-03 1号/2号隧道
- **参数**：1号隧道 = EMA144 + EMA169；2号隧道 = EMA288 + EMA338。
- **判定**：价格站上1号隧道偏强；回踩1号确认支撑是买点；跌到2号是深度支撑/加仓观察位。
- **行动**：跌破1号后等2号止跌；打穿1号/2号后重新站回2号才可考虑买。

### EMA-04 EMA+MACD+隧道三合一
- **买入条件**：MACD金叉 + EMA13上穿EMA21 + K线实体站上1号隧道。
- **卖出/回调条件**：MACD死叉 + EMA13下穿EMA21 + K线跌破隧道。
- **行动**：三条件同步时信号有效；不同步时等待。

## 5. KDJ规则

### KDJ-01 情绪区间
- **量化判定**：K>80或J>100 = 超买；K<20或J<0 = 超卖。
- **行动**：超买警惕回调，超卖关注反弹。
- **失效条件**：强趋势中KDJ可能钝化，必须结合结构。

### KDJ-02 震荡买卖点
- **触发条件**：标的处于整理/震荡区间。
- **买入**：K<30或接近30时，K线上穿D线形成金叉。
- **卖出**：K>70时，K线下穿D线形成死叉。
- **行动**：KDJ只做节奏，不单独推翻大趋势。

## 6. MACD规则

### MACD-01 五句口诀
1. 零上金叉 → 强势上涨。
2. 零上死叉 → 看回调。
3. 零下金叉 → 只当反弹。
4. 零下死叉 → 弱势看跌。
5. 零轴本身是支撑/压力。

### MACD-02 能量柱顿化
- **量化判定**：红/绿柱远离零轴＝能量增强；靠近零轴＝能量减弱；红绿柱交替形成板型/顿化＝暂停或反转领先信号。
- **行动**：顿化时可提前减仓或等待确认，不必等死叉才反应。

## 7. 布林带规则

### BOLL-01 标准循环
- **路径**：突破上轨走加速 → 回落中轨支撑 → 下轨支撑 → 震荡 → 再加速。
- **判定**：跌破中轨且下轨也撑不住 → 加速下跌；碰下轨撑住 → 反弹观察。

### BOLL-02 比特币周线定投框架
- **买入**：周线触及布林带下轨并验证支撑。
- **加仓**：回踩布林带中线不破。
- **卖出**：触及上轨或上涨后横盘无力。
- **注意**：BTC优先看大周期，分钟/日内易插针。

## 8. 成交量/成交额规则

### VOL-01 量额背离判断假K线
- **上涨日**：成交量平量/微升但成交额极度缩量 → 阳线有水分，主力未参与，不追高。
- **下跌周/月**：成交量放大或平量但成交额缩量 → 可能是假阴线/拐点，需要后续确认。
- **失效条件**：次日/次周成交额与成交量同步放大验证方向。

### VOL-02 量额共振确认真信号
- **量化判定**：成交量与成交额同向放大，且跨指数/板块同步放量。
- **行动**：提高突破/阳线可信度。
- **反例**：只有部分指数放量，非共振上涨，次日更可能回踩震荡。

## 9. 形态规则

### PAT-01 三重底
- **触发条件**：三次下探同一支撑区域。
- **量化判定**：三个低点基本持平，第三个不破前低。
- **行动**：第三低点确认后可买/持有。
- **失效条件**：跌破前低。

### PAT-02 假双顶/假双底
- **触发条件**：形态看似双顶/双底。
- **量化判定**：第二次触及时成交萎缩，未真正跌破/突破关键位。
- **行动**：假双顶视为诱空可反向看涨；仍需关键位确认。

### PAT-03 头肩底目标
- **量化判定**：头部到颈线垂直距离=X；突破颈线后目标=颈线+X。
- **失效条件**：跌破右肩。

### PAT-04 牛旗
- **触发条件**：快速拉升后窄幅整理，波动率降低。
- **行动**：整理末端突破买入/加仓。

### PAT-07 楔形突破后的隧道回踩旗形
- **触发条件**：长期震荡/楔形整理结束后，价格突破并上攻隧道，随后回踩再度上攻隧道。
- **量化判定**：至少出现“突破整理区 → 冲击隧道 → 回踩不破 → 再次冲击隧道”四步，且整理形态向上倾斜或收敛。
- **行动**：视为上升旗形/多头蓄力，等待再次放量上攻；未突破前只观察。
- **失效条件**：回踩跌回原整理区或跌破旗形下沿。

### PAT-05 五浪力度递减
- **触发条件**：月线/周线5浪推动末期。
- **量化判定**：第3浪小于第1浪，第5浪无力创新高。
- **行动**：视为见顶/减仓信号。

### PAT-06 熊市打折
- **触发条件**：资产牛市顶后进入熊市。
- **量化判定**：顶部价×50%作为熊市目标参考。
- **行动**：到目标附近才考虑底部机会，不在高位硬接。

## 10. 市场/品种专用规则

### GOLD-01 黄金关键位
- **3890规则**：站稳3890有效突破 → 看3950第一目标，再看4030；站不稳 → 下探3836，破则看3800。
- **4200/4000规则**：4200不破可视为ABC调整后延续；破4200看4000。
- **4500/5050远期**：斐波那契拓展用于压力/目标验证，不能当确定预测。

### GOLD-02 黄金与美元
- **判定**：避险强时美元和黄金可同涨；避险消退后美元反弹通常压制黄金。
- **美元指数**：若攻击并站稳2号隧道上沿，对黄金有强回调作用；美元靠近100时黄金可能见底部区域。

### GOLD-03 黄金开盘30分钟
- **触发**：XAUUSD每日开盘前30分钟。
- **判定**：前30分钟上涨 → 当天更可能回调；前30分钟低位震荡 → 当天更可能反弹。
- **失效**：重大数据/事件。

### GOLD-04 黄金主升浪EMA8/13支撑
- **触发**：黄金处于日线级别主升/加速段，价格回踩短期均线。
- **判定**：若日线收盘仍站在EMA8/EMA13上方或两者之间，趋势未走坏，不因单次回踩提前看空。
- **行动**：趋势未破时以持有/观察为主；回踩EMA8-EMA13不破可视为强势整理。
- **失效**：日线连续跌破EMA13并跌向隧道支撑，或跌破关键通道/结构支撑。

### GOLD-05 通道上沿突破与隧道回踩双路径
- **触发**：黄金逼近上行通道上沿或关键结构压力。
- **判定**：突破并站稳通道上沿 → 看下一结构压力；反复受阻不过 → 优先看下方隧道/均线支撑。视频例：站稳约5130后看约5560；若不过则看约4780-4800隧道区。
- **行动**：突破站稳才升级为右侧确认；未突破前不追，回踩支撑再评估。
- **失效**：假突破后跌回通道内，或回踩隧道区失守。

### GOLD-06 美元指数斐波那契/支撑带反向提示
- **触发**：美元指数接近关键斐波那契压力/支撑或大周期支撑带。
- **判定**：美元在双重斐波那契压力处受阻，有利于黄金支撑；美元在大周期支撑带止跌并站稳上沿，则提示黄金回调风险。
- **行动**：黄金判断需同步检查美元指数；美元未止跌时不轻易判定黄金深跌，美元确认反弹时降低黄金追多等级。
- **失效**：美元跌破支撑带或突破压力后的走势没有得到黄金反向验证。

### ETF-01 周期股/ETF
- **适用**：有色、化工、钢铁等周期板块。
- **行动**：用ETF分散风险，分批建仓，不压单只个股，不一把梭。

### ETF-02 恒科/中概配置
- **稳健**：恒科ETF作为底仓，中概ETF博弹性。
- **激进**：中概ETF多配，恒科ETF降波动。
- **普通**：5/5或6/4均衡。

### ETF-03 信创替代
- **方法**：计算机ETF + 软件ETF = 1:1。
- **逻辑**：比信创ETF更纯粹，位置更低，风险收益比更好。

### CRYPTO-01 双均线通道
- **开多**：站上通道下轨并验证。
- **止损**：跌破通道下轨。
- **开空**：下跌趋势中反抽验证通道上轨。
- **事件性波动**：止损不能过小，否则容易被插针洗出。
