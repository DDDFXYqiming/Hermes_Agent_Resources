## 一、成交量/成交额共振与背离（真伪K线判断）

### 规则 1：量-额背离判断K线真伪

| 维度 | 内容 |
|------|------|
| **触发条件** | 日线/周线/月线收盘后，成交量（手/首）与成交额（元）出现方向不一致 |
| **量化判定** | **情形A（上涨日）**：成交量平量微升（略微放量） + 成交额极度缩量 → 该阳线是假的（散户推动，主力未参与），有水分<br>**情形B（下跌周）**：成交量平量微缩 + 成交额缩到黄线之下 → 周阴线是假性阴线（真阳线/拐点）<br>**情形C（月线）**：成交量绿色放量 + 成交额缩量 → 月K阴线是假性阴线 |
| **行动** | 情形A→下个交易日不放量则预期回调，不追高；情形B→看涨拐点；情形C→月线假阴线 |
| **失效条件** | 次日/次周放量验证则推翻假性判断 |
| **证据** | **BV1dbUoBzEqD**「成交量/成交额的共振/背离判断真假性K线：上证指数、恒生科技、科创芯片、科创50」<br>• 00:11-00:24："成交量是平量略微放量，而成交额是极度的一个缩量……主力几乎没有参与……这种背离情况，就证明当天的阳线是有点水分的"<br>• 结构化摘要00:00-02:09："成交量平量微缩…成交额又缩到了黄线之下…量平但额缩这种背离情况，就证明上一周的一个阴线，是一个假性的阴线"<br>• "成交量放量一个跌幅，然后成交额呢就是一个缩量…成交额去萎缩，但是成交量又去放大…月线也是个假的，假性的一个月线" |

## 三、斐波那契回调（黄金分割线）

### 规则 5：0.618 — 国际资产趋势定义位

| 维度 | 内容 |
|------|------|
| **触发条件** | 国际资产（黄金、白银、外汇等）处于趋势回调阶段 |
| **量化判定** | • **上涨趋势**：回调不能跌破0.618；若跌破→趋势反转（不再是上涨趋势）<br>• **下跌趋势**：反弹不能涨破0.618；若涨破→趋势反转（不再是下跌趋势）<br>• 0.618在上涨中构成强烈支撑，在下跌中构成强阻力 |
| **行动** | 0.618守住→维持原趋势判断；0.618被日线级别突破→重新判断趋势方向 |
| **证据** | **BV1puCgBuE7e**「神奇的斐波那契回调：黄金分割线应用」<br>• 结构化摘要00:00-02:38："0.618是定义一个趋势非常重要的一个位置……如果是上涨的一个趋势，它不能跌破，跌破的话那就不只是一个上涨趋势了，就会反转趋势"<br>• "0.618在上涨当中会有一个很强烈的支撑，下跌的时候0.618又会有一个很强的阻力" |

### 规则 10：MACD能量柱 + 顿化信号

| 维度 | 内容 |
|------|------|
| **量化判定** | • 能量柱（红绿柱）**远离零轴** → 能量增强<br>• 能量柱**靠近零轴** → 能量减弱/暂停/反转<br>• 红绿柱交替形成「板型」→ **顿化现象**，是趋势暂停或反转的**领先信号**（领先于死叉出现） |
| **行动** | 出现顿化→提前减仓或注意反转风险（不必等死叉） |
| **证据** | **BV1Xz1QB2Eox** 结构化摘要10:17："能量柱越远离零轴时候，证明它能量的逐渐增加强；越靠近零轴时候证明能量减弱……顿化的话呢，就要看能量是否慢慢地再减弱，也是一种暂停或是反转的一种信号，它会领先于死叉" |

### 规则 11：布林带运行模式

## 五、压力位/支撑位三次验证

### 规则 12：压力位「不过三」原则

| 维度 | 内容 |
|------|------|
| **触发条件** | 价格反复测试同一压力位 |
| **量化判定** | 一个压力位被测试**3次**之后 → 压力位**失效**，第4次很容易突破（"不过三"） |
| **行动** | 第3次测试压力位 → 预期突破概率高，可布局突破方向 |
| **失效条件** | 若第3次仍然压制住则压力位仍然有效 |
| **证据** | **BV1WE1QBwEiX**「黄金多空激战，胜负即将分晓」<br>• 结构化摘要00:00-01:50："这种压力位……压力位一般都是不过3的，破3次，1,2,3，这一次他一定要破过去才可以……一个压力位只要验证3次之后，一般就都失效了，就可以非常容易的破过去" |

### 规则 13：支撑/压力位转换

| 维度 | 内容 |
|------|------|
| **量化判定** | • 压力位被突破并**站稳** → 变为支撑位<br>• 支撑位被跌**破** → 变为压力位 |
| **证据** | **BV1XTkqBhE6s** 结构化摘要04:34："3条压力8321……当他跌穿之后，然后支撑就变压力，然后每到碰到上面，就会产生一个回调"<br>**BV19gUyBPEvk** 结构化摘要00:59："如果收线的话，它站在这个压力之上，压力变支撑的话，那么可以视为呢，是个短线走多的一个表现" |

## 九、跨视频验证的通用原则

1. **真假K线判断链**：量额背离（日/周/月）→ 判断K线水分 → 决定追高/等回调
2. **买入决策链**：看位置（通道/箱体/斐波那契）→ 看均线（隧道支撑/短期均线）→ 看MACD → 三合一买入信号 → 右侧确认
3. **卖出/减仓链**：MACD死叉 + EMA死叉 → 压力位测试3次不过 → 量额背离阳线 → 减仓
4. **趋势反转链**：0.618被突破（国际）或0.382被跌破（国内）+ MACD零轴行为 → 判断趋势是否反转
5. **时间换空间**："输时间不输钱"—横盘震荡用时间消耗筹码，不一定要杀空间（BV1WE1QBwEiX）
