# 证据与口径说明

> 本文件只保留公开复用所需的证据口径，不记录本机路径、个人目录、临时工作区、用户账号标识或过细来源细节。

## 资料口径

- 方法论来源：用户提供的市赚率整理材料、丁宁公开投资文章，以及公开财务资料的交叉核验。
- 本 skill 不保存原始材料路径；如需追溯原始文档，请由当前使用者在自己的知识库中定位。
- 公开版只保留方法规则、案例口径和核验要求，不保存 UID、主页、临时下载目录、截图目录等可识别信息。

## 方法论主题覆盖

- 市赚率三公式：稳定 ROE 企业、周期/资源资产、指数/ETF。
- 股利支付率修正：用分红率差异修正 PR 阈值。
- A/H 股与红筹税负：税后现金流会影响合理估值带。
- 经典案例：喜诗糖果、可口可乐、中国石油、华盛顿邮报、苹果、伯克希尔公开持仓案例。
- 资产轮动：用估值相对便宜程度筛选红利、债券、商品等资产，而不是机械套用宏观时钟。

## 历史事实使用原则

| 类型 | 使用方式 |
|---|---|
| 历史案例数据 | 可作为方法论示例；输出时标明“历史案例/非当前行情”。 |
| 当前行情与估值 | 必须重新查询并写明数据日期。 |
| 作者个人测算 | 可作为解释口径；不能替代公告、财报、交易所或基金公司资料。 |
| 搜索摘要 | 只能作为线索；关键数字要回到公告、财报、基金资料或可靠行情源。 |

## 置信度标记

- `verified`: 已用公告、财报、基金资料、交易所披露或可靠行情源核验。
- `method-example`: 用于说明方法的历史案例，不代表当前可买卖。
- `stale`: 历史数据可引用，但当前分析需更新。
- `interpretive`: 方法论解释或经验总结，不是硬事实。
