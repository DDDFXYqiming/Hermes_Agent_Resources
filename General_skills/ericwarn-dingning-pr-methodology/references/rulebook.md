# 丁宁市赚率(PR)规则书

> 来源：用户整理的 `C:\Users\39795\Desktop\books\雪球\PR.md`、雪球 ericwarn丁宁 文章合集、以及联网核验后的公开资料。用途：复现和审计丁宁的估值框架，不构成投资建议。

## 1. 方法论总纲

1. **先估值，后择时**：市赚率回答“值不值得买”，技术分析只回答“何时执行”。
2. **低估不是买入充分条件**：必须检查 ROE 质量、分红真实度、周期阶段、税负和数据来源。
3. **4折5折6折不是预测收益**：它是买入价格相对经营回报的安全边际描述。
4. **A股与H股分开估**：长期股息税和红筹税负会直接压低合理PR阈值。
5. **周期股看正常化ROE**：不能拿景气顶点或低谷单年ROE直接计算。

## 2. PR公式选择

### PR-01 第一公式：稳定ROE企业
- **公式**：`PR = PE / (ROE × 100)`。
- **适用**：贵州茅台、可口可乐、喜诗糖果等 ROE 长期稳定且盈利质量高的企业。
- **输入**：PE-TTM 或合理年化PE；ROE-TTM 或多年平均ROE。
- **动作**：计算后映射到 0.4/0.5/0.6/1PR 区间。
- **失效条件**：ROE异常、一次性利润、商业模式恶化、分红政策大变。

### PR-02 第二公式：周期股/资源股
- **公式**：`PR = PB / (ROE × ROE × 100)`。
- **适用**：中国石油、中国海油、中国神华、煤炭、石油、有色等周期资产。
- **输入**：当前PB + 多年或景气周期第一年ROE均值。
- **动作**：用PB承接资产折价，用正常化ROE过滤周期波动。
- **失效条件**：用景气顶点ROE、忽略商品价格中枢、资产减值风险未处理。

### PR-03 第三公式：指数基金/ETF
- **公式**：`PR = PE × PE / (PB × 100)`。
- **适用**：159209红利质量ETF、红利指数、宽基或行业指数。
- **输入**：指数/ETF PE 与 PB；若可能，补充成分股质量和调仓机制。
- **动作**：低于0.7-0.8PR可进入观察/定投区，低于0.6PR提高跟踪等级。
- **失效条件**：指数编制变化、成分股质量恶化、估值口径不一致。

### PR-04 修正市赚率：股利支付率修正
- **公式**：`PR' = N × PE / (ROE × 100)`，`N = benchmark payout / actual payout`。
- **丁宁原始标杆**：以茅台历史约50%股利支付率为标杆。
- **新事实**：茅台2024-2026已将分红率提升至不低于75%，2024实际约75%，2025约79%。分析最新个股时必须使用最新分红政策。
- **适用**：银行、保险、国央企、分红稳定但留存收益质量差异大的企业。
- **动作**：股利支付率越低，估值折扣越大。
- **失效条件**：成长股需要留存再投资、科技股以回购代替分红、周期股景气期临时高分红。

## 3. 估值区间规则

| PR区间 | 标签 | 解释 |
|---|---|---|
| `<0.4PR` | 4折极低估 | 类可口可乐/中石油级安全边际，重点跟踪但仍需风控 |
| `0.4-0.5PR` | 好球区 | 巴菲特式“40-50美分买1美元” |
| `0.5-0.6PR` | 6折击球区 | 可轻仓/分批研究，需确认质量与趋势 |
| `0.7-0.8PR` | ETF观察区 | 指数基金可定投/观察，个股需谨慎 |
| `≈1PR` | A股合理/高估阈值 | 丁宁体系里A股常用高估阈值 |
| `H股≈0.8PR` | H股合理/高估阈值 | 20%股息税折算后的阈值 |
| `红筹H≈0.72PR` | 红筹更低阈值 | 28%左右税负折算 |

## 4. 丁宁核心案例基准

| 案例 | 关键数据 | PR意义 |
|---|---|---|
| 喜诗糖果 1972 | 2500万美元收购，净利约208-210万，净资产约800万，PE约12，ROE约26% | 约0.5PR，优秀企业半价买入 |
| 可口可乐 1988-1989 | 丁宁测算14.7-15PE、31-46% ROE | 两年平均约0.4PR |
| 中国石油H 2003 | 4.88亿美元、0.208-0.209美元/股、正常ROE约15.88% | 约0.38PR，周期股第二公式样板 |
| 华盛顿邮报 1973 | 1060万美元成本，市值8000万 vs 内在价值约4亿 | 特许经营权折价样板，PR需结合商业本质 |
| 苹果AAPL 2016 | 93-122美元建仓，约12-14PE、高ROE、高回购 | 科技股中罕见可用PR估值的例外 |

## 5. 资产配置规则

### ASSET-01 “啥便宜买啥”替代美林时钟
- **触发条件**：宏观周期难以判断、资产相关性混乱。
- **动作**：不用先猜周期，优先比较不同资产估值：红利基金、美国国债、原油/商品等。
- **失效条件**：资产本身现金流或政策约束不可比。

### ASSET-02 红利基金平替高估银行股
- **逻辑**：当银行股修正PR普遍超过1PR，而红利ETF约0.7-0.8PR且有自动调仓机制时，用红利ETF替代单一银行股。
- **验证点**：ETF跟踪指数、分红频率、成分股ROE质量、费用率、流动性。

### ASSET-03 159209第三公式样板
- **指数逻辑**：红利筛选 + 质量优选。
- **关键指标**：连续分红、股利支付率区间、现金股息率、ROE、ROE稳定性、经营现金流/负债。
- **注意**：指数编制规则和成分权重会变，必须查最新资料。

## 6. 风险否决规则

1. **数据未核验否决**：PE/PB/ROE/payout不是同一日期或来源不明，标记“数据不足/不计算”。
2. **ROE失真否决**：一次性收益、负净资产、超高杠杆导致ROE虚高，需改用ROA或PB/正常化ROE。
3. **分红质量否决**：低分红且留存收益无法创造更高ROE，修正PR必须上调。
4. **周期误判否决**：资源股必须确认商品价格中枢和周期位置。
5. **税负忽略否决**：A/H股、红筹、陆股通、ETF分红税差必须写明。
6. **交易执行否决**：估值低但技术结构破坏时，不直接给买入，只给观察或分批计划。
