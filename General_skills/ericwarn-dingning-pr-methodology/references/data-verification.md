# 数据验证清单

## 必须联网/实时验证的数据

1. 当前股价、市值、PE-TTM、PB、ROE-TTM。
2. 最新年报/季报的ROE、净利润、净资产、分红率。
3. ETF跟踪指数、编制规则、成分股、费率、分红记录。
4. 13F、公告、回购、股息税、政策变化。
5. 债券收益率、商品价格、汇率等宏观输入。

## 推荐来源优先级

| 类型 | 优先来源 |
|---|---|
| 美股13F | SEC、13F.info、13Radar、Morningstar、StockTitan |
| A股行情/估值 | 东方财富、理杏仁、同花顺、证券交易所公告、公司年报 |
| 港股 | 港交所公告、Yahoo Finance、同花顺港股、新浪港股 |
| ETF | 基金公司公告、产品资料概要、天天基金、交易所 |
| 税制 | 国家税务总局、财政部、证监会公告 |
| 宏观利率 | 中债估值、中国货币网、TradingEconomics、CEIC |

## 核验规则

- 同一数值至少查 2 个来源；若只有一个来源，降低置信度。
- 区分丁宁原文数据与当前数据：原文数据可原样引用，但当前分析必须更新。
- 把日期写进输出：例如“2026-06-17 PE-TTM 18.74”。
- 不同口径冲突时，说明：TTM/静态/扣非/摊薄/年化。
- 不得用搜索摘要里的数字直接替代公告原文，除非任务不需要精确审计。

## 常见时效陷阱

- 茅台分红率：丁宁常用50%历史标杆，但2024-2026公告已提高到不低于75%。
- 巴菲特退休：正式卸任CEO是2025-12-31，Greg Abel 2026-01-01接任；2026-05-02是Abel首次主持股东大会。
- 13F：报告期、披露日、交易发生期不同；2025Q4 13F披露日为2026-02-17。
- ETF指数规则会调整，必须查最新产品资料概要。
