# PR估值卡模板

| Field | Content |
|---|---|
| 标的 | 代码/名称/市场 |
| 资产类型 | 稳定ROE个股 / 周期股 / 指数ETF / 银行保险 / H股红筹 / 其他 |
| 数据日期 | YYYY-MM-DD |
| 数据来源 | 来源1；来源2；公告/年报链接 |
| 使用公式 | PR1 / PR2 / PR3 / 修正PR |
| 输入数据 | PE=；PB=；ROE=；股利支付率=；税负= |
| 原始PR | 计算式与结果 |
| 修正PR | N=；计算式与结果；如不适用说明原因 |
| 估值区间 | 4折 / 5折 / 6折 / 观察 / 合理 / 高估 |
| 质量检查 | ROE质量；现金流；分红；杠杆；周期位置 |
| 风险否决 | 数据缺失/ROE失真/周期误判/税负/政策/技术结构 |
| 与fox配合 | 是否需要技术确认；关键支撑/压力待查 |
| 结论标签 | 数据不足/不计算 等8个允许标签之一 |
| 非投资建议 | 明确说明仅为方法论复盘/学习 |

## 简短输出格式

```md
### PR估值卡：<标的>

- 数据日期：
- 公式选择：
- 输入：PE=，PB=，ROE=，股利支付率=，税负=
- 计算：
- 估值区间：
- 风险否决：
- fox执行检查：
- 标签：
- 说明：仅为方法论复盘，不构成投资建议。
```
